# Workflow recomendado en Antigravity

## Antes de cada módulo

Pedir al agente:

```txt
Antes de implementar, inspeccioná el código existente, proponé un plan corto y confirmá que no vas a tocar funcionalidades fuera del módulo.
```

## Después de cada módulo

Pedir:

```txt
Revisá lo implementado, buscá errores, indicá comandos para migraciones/tests, y no agregues nuevas funcionalidades.
```

## Si el agente intenta avanzar de más

Responder:

```txt
No avances con esa funcionalidad todavía. Limitate al módulo actual. Dejalo como TODO documentado si corresponde.
```

## Si hay errores

Pedir:

```txt
Corregí únicamente los errores relacionados con el módulo actual. No refactorices partes no relacionadas.
```
